# Soumya Ranjan Das - Portfolio Website

This is a modern, animated single-page portfolio website showcasing the work of Soumya Ranjan Das, a professional graphic designer from Odisha, India.

## Features

- Responsive single-page design
- Smooth animations and transitions
- Dark/Light mode toggle
- Interactive skill bars
- Animated project cards
- Contact form
- Back to top button
- SEO optimized

## Setup

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. No build process required - it's a static site!

## Structure

```
CV PORT FOLIO/
├── index.html
├── styles.css
├── script.js
├── README.md
└── assets/
    ├── background-animation.json
    └── [project images]
```

## Customization

To add your own projects:
1. Add project images to the `assets/` folder
2. Update the projects array in `script.js`
